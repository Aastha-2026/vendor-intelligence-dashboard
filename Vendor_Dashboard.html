📊 How to use RFP Builder

Build a competitive RFP in 3 simple steps:

☑
Step 1 — Select vendors
On any vendor card, tick the Select for RFP checkbox (top-right corner). Select as many vendors as needed.
📄
Step 2 — Generate template
A blue bar appears at the bottom showing selected vendors. Click Generate RFP Template to create a pre-filled RFP document.
📋
Step 3 — Copy & send
Copy the generated RFP template and paste into Teams, Outlook, or Word to send to vendors for competitive bidding.
